# whitespace-quine

Generate a Quine in Whitespace to the following files:

- `quine.ws.asm` - Whitespace Assembly code
- `quine.ws` - Assembled code

## Pre-requisites

- Docker
- Python 3.10 or later
- Poetry 2.1.2 or later

## Development

Run `make help` for details.
